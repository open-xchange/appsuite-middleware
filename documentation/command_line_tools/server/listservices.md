# NAME

listservices - List services

# SYNOPSIS

**listservices** [OPTIONS]


# DESCRIPTION

This command line tool lists the registered services.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password
